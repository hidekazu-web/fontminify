## 課題詳細

### ISS-003: 型チェックエラー（ビルドブロッカー）
- **優先度**: P0 (緊急)
- **カテゴリ**: ビルド
- **説明**: `npm run typecheck` で5件のTypeScriptエラーが発生し、コンパイルが通らない
- **エラー内容**:
  1. `fontkit` の型定義がない（fontSubsetter.ts:167）
  2. `File.path` が存在しない（FileDropZone.tsx:40, 46）
  3. 型の不一致 `string` → `"ttf"|"otf"|...`（fontStore.ts:174, 204）
- **影響範囲**: ビルド全体がブロックされる
- **対応方針**:
  1. `@types/fontkit` をインストールまたは型定義ファイル作成
  2. Electron環境でのFile.path対応を修正
  3. fontStore.tsの型キャストを修正

### ISS-004: テスト実行問題
- **優先度**: P1 (重要)
- **カテゴリ**: テスト
- **説明**: `npm test` 実行時にテストが完了しない（ハングまたは長時間実行）
- **影響範囲**: CI/CDパイプライン、品質保証
- **対応方針**: テスト設定の確認、タイムアウト設定の見直し

### ISS-001: ダークモードテスト
- **優先度**: P2 (低)
- **カテゴリ**: UIテスト
- **説明**: macOSシステム設定のダークモード切り替え時にアプリが正しく追従するかのテストが未実施
- **影響範囲**: UI表示のみ、機能には影響なし
- **対応方針**: 手動テストまたは自動テスト追加

### ISS-002: Retina対応テスト
- **優先度**: P2 (低)
- **カテゴリ**: UIテスト
- **説明**: 高解像度ディスプレイでのUI表示品質テストが未実施
- **影響範囲**: UI表示のみ、機能には影響なし
- **対応方針**: Retinaディスプレイ環境での目視確認
