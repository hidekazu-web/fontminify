# FontMinify アイコン仕様

## Retina対応アイコンサイズ

### macOS アプリケーション用アイコン (.icns)
- 16x16 (1x) / 32x32 (2x)
- 32x32 (1x) / 64x64 (2x)  
- 128x128 (1x) / 256x256 (2x)
- 256x256 (1x) / 512x512 (2x)
- 512x512 (1x) / 1024x1024 (2x)

### 必要ファイル
- icon.icns (macOS用メインアイコン)
- icon@2x.icns (Retina対応)
- icon.png (512x512 - 開発用)
- icon@2x.png (1024x1024 - Retina開発用)

### デザイン要件
- シンプルで視認性の高いデザイン
- FontMinifyの「フォント最適化」を表現
- macOSのデザインガイドラインに準拠
- ダークモード/ライトモード両対応

### カラーパレット
- プライマリ: #667eea (primary-500)
- セカンダリ: #764ba2 (secondary-500) 
- アクセント: グラデーション
- テキスト: 白/黒（コントラスト対応）

### アイコンコンセプト
1. 「A」文字 + 最適化アイコン（矢印、圧縮表現）
2. フォントファイル + 軽量化表現
3. 日本語フォント（ひらがな等）+ 最適化表現
